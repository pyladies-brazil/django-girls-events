Bem Vind@!

Se você está por aqui é porque possui algum interesse pelo Django Girls. (e isso é demais!)

Django Girls é uma instituição sem fins lucrativos. Nossa missão é iniciar mulheres na área de tecnologia. Para saber mais sobre, acesse a página oficial da organização.

Este projeto é um agregador de informações para eventos no Brasil. Sinta-se livre para melhorá-lo ou adicionar o seu evento aqui! Para mais informações acesse o nosso repositório no GitHub.

Você pode nos ajudar de muitas maneiras! Faça um fork deste projeto, ajude-nos a manter a nossa documentação pt-br ou crie o seu próprio evento Django Girls na sua cidade! Para mais informações, acesse a seção Documentação
