Documentação

Nem toda a documentação do Django Girls se encontra atualmente traduzida para o pt-br. Caso você tenha interesse em ajudar com isso nós seríamos eternamente gratas <3 Faça o fork de algum repositório de documentação do Django Girls e nos avise. Nós podemos te ajudar com esta missão e assim que for finalizada vamos adicioná-la aqui.

Atualmente três documentos estão completamente traduzidos para o pt-br mas sempre estão em constante atualização:

Django Girls Tutorial

O Tutorial é a base dos nossos workshops. Completamente voltado para iniciantes, ele possui informações sobre linha de comando, Python, Django, HTML e CSS. Você pode acessá-lo neste link. Para ajudar nesse projeto de tradução basta ter uma conta no Crowdin e acessá-lo aqui.

Manual d@ Organizador@

Este Manual é uma mão na roda para quem quer iniciar um evento Django Girls. Ele possui praticamente todas as informações possíveis para isto! (coletadas e atualizadas com as experiências de todos os eventos passados) Para ajudar neste projeto, faça um fork do nosso repositório e submeta a tradução através de pull request.

Manual d@ Treinador@

Não se deixe enganar: Guiar iniciantes pelo caminho do conhecimento não é uma tarefa fácil! Este manual irá auxiliar @s treinador@s nesta tarefa. Para ajudar neste projeto, faça um fork do nosso repositório e submeta a tradução através de pull request.
